什么是响应式布局

一个网站能够兼容多个终端,并且在各个终端都可以有很好的展示体验

bootstrap

  • 栅格参数
  • .col-xs- 超小屏幕手机<768px>
  • .col-sm- 小屏幕 平板(>=768px)
  • .col-md- 中等屏幕 桌面显示器(>=992px)
  • .col-lg- 大屏幕 大桌面显示器(>=1200px)
  • 列偏移
  • .col-md-offset-* 类可以将列向右侧偏移
  • .col-md-push-*和.col-md-pull-*
  • 显示隐藏
  • visible-md,hidden-md

媒体类型

  • all所有媒体
  • braille盲文触觉设备
  • embossed打印机
  • print手持设备
  • projection打印预览
  • screen彩屏设备
  • speech"听觉"类似的媒体类型
  • tty不适用像素的设备
  • tv电视

关键字

  • and 连接
    • @media all and(min-width:1201px)
  • not
    • 关键字是用来排除某种特定的媒体类型
    • @media not
  • only
    • 某种特定的媒体类型
    • @media only tv

Css3响应式布局

  • 媒体特性
  • width文档宽度
  • orientaion 横竖屏(portrait/landscape)
  • @media()

row

  • 行(row),必须包含在.container(固定宽度)或.container-fluid(100%宽度)中

栅格参数

  • .col-xs- 超小屏幕手机
  • .col-sm- 小屏幕 平板(>=768px)
  • .col-md- 中等屏幕 桌面显示器(>=992px)
  • .col-lg 大屏幕大桌面显示器(>=1200px)
  • 列偏移
    • .col-md-offset*:类可以将列向右侧偏移
    • 列排序:.col-md-push-*和.col-md-pull-*
    • 显示隐藏visible-md,hidden-md
